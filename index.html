Wind Triangle Calculations

Wind Correction Angle (WCA):
WCA = arcsin((Wind Speed / TAS) × sin(Wind Angle))
where Wind Angle = Wind Direction - Track

Magnetic Heading:
True Heading = Track - WCA
Magnetic Heading = True Heading - Magnetic Variation

Ground Speed:
GS = TAS × cos(WCA) - Headwind Component
where Headwind Component = Wind Speed × cos(Wind Angle)
(Positive for headwind, negative for tailwind)

Note: All calculations use the wind triangle method. A crosswind requires crabbing into the wind (WCA) to maintain track, which slightly reduces ground speed. Headwinds reduce ground speed, tailwinds increase it.
